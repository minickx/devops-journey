# Docker

Exercises and assignments completed during [Docker Mastery](https://www.udemy.com/course/docker-mastery/) by Bret Fisher.

## Contents

### `basics/`
Core concepts: images vs containers, run/exec differences, networking, and container inspection.

### `multi-container-assignment/`
Running and managing multiple containers simultaneously (nginx, httpd, MySQL) — port mapping, naming, and env vars.

### `entrypoint/assignment01/`
Building custom images with ENTRYPOINT scripts. Two variants: apachebench and cmatrix.

### `dockerfile-entrypoint/`
Dockerfile using ENTRYPOINT with a startup script for a Python/FastAPI app.

### `compose-assignment-drupal/`
Multi-container app with Docker Compose — Drupal CMS + PostgreSQL, volumes, environment variables.
